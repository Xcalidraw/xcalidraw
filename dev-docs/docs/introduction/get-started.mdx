## Try now

Go to [xcalidraw.com](https://xcalidraw.com) to start sketching.

## How are these docs structured

These docs are focused on developers, and structured in the following way:

- [Introduction](/docs/) — development setup and introduction.
- [@xcalidraw/xcalidraw](/docs/@xcalidraw/xcalidraw/installation) — docs for the npm package to help you integrate Xcalidraw into your own app.
- Editor — IN PROGRESS. Docs describing the internals of the Xcalidraw editor to help in contributing to the codebase.
- [@xcalidraw/mermaid-to-xcalidraw](/docs/@xcalidraw/mermaid-to-xcalidraw/installation) - Docs for the mermaid to xcalidraw parser
