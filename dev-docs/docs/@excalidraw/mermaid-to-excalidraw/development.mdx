# Development

This page relates to developing the `@xcalidraw/mermaid-to-xcalidraw` package itself.

### Clone the Repository

Go to [@xcalidraw/mermaid-to-xcalidraw](https://github.com/xcalidraw/mermaid-to-xcalidraw) and clone the repository to your local.


```bash
git clone git@github.com:xcalidraw/mermaid-to-xcalidraw.git
```

### Install the dependencies

Using `npm`

```bash
npm install @xcalidraw/mermaid-to-xcalidraw
```

Using `yarn`

```bash
yarn add @xcalidraw/mermaid-to-xcalidraw
```

### Run the playground server

```bash
yarn start
```

This will start the playground server in port `1234` and you start playing with the playground.

## Creating a test release

We will soon simplify creating release via commenting on GitHub PR similar till then you can create a release by following the below steps

1. Create the build

```bash
yarn build
```

This will create the dist folder which we need to publish next.

2. Publish the library

Update the package name and version in [package.json](https://github.com/xcalidraw/mermaid-to-xcalidraw/blob/master/package.json) and run the below command to publish it

```bash
yarn publish
```
